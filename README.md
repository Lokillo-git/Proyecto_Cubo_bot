# Proyecto_Cubo_bot
Proyecto de Cubo Bot educativo desarrollado con Arduino. El repositorio contiene código fuente para control mediante tarjetas RFID RC522, grabación de etiquetas MIFARE Classic, control de motores, versiones con conectividad Bluetooth, además de los archivos de diseño y fabricación de PCB (Gerber y FlatCAM) necesarios para la construcción del robot.

## Características

- Lectura de tarjetas RFID RC522
- Grabación de tarjetas MIFARE Classic
- Control de motores y movimientos programados
- Versión con comunicación Bluetooth
- Archivos PCB para fabricación
- Compatible con Arduino Nano

## Estructura

- Codigo_Cubo_Bot_RFID/
- Codigo_Cubo_Bot_bLUTUD/
- Codigo_Grabar_tarjetas_Mismo_Circuito/
- Archivo_GRBControl_FlatCam_*/

## Hardware utilizado

- Arduino Nano
- RC522 RFID
- Motores DC con driver
- Buzzer
- Tarjetas MIFARE Classic
